A1.sync

即使一个人做了错事犯了法，也只有法律才有定罪的资格。

事实上，法律作为一种底层能力，最重要的价值正是风险防范。

程序员非法出售 VPN，被判刑三年并罚款一万 男子卖“绝地求生”外挂月入数万，最后却被抓 腾讯 23 岁工程师检查酒店 WiFi 漏洞，被新加坡安全局逮捕

像是给私人建设黄色网站、帮人搭建套现平台这些活儿，甚至只是“爬虫”获取一些数据信息，他们也会注意不能“过了线”。

比如在刚入职签劳动合同时，我会仔细阅读工资待遇、五险一金和保密事项等内容，因为我清楚，一旦落笔，这张纸便成了我板上钉钉的法律责任，即使有不合理的地方也得认了。

再如租房时，我会查清楚中介的资质和房东的手续，正式入住之前也会记录好所有大件物品的交接，然后严肃对待租房合同，这样在出问题时，才知道怎么保护自己的利益

参与、思考并输出，才是学习的最佳途径。
